Converting from ternary to balanced ternary

Question. The purpose of this exercise is to write a C program which reads on the command-line a string representing an integer in ternary form and outputs its corresponding balanced ternary representation.

Hints and constraints.
  • The input must be checked in the main function, before the computation takes place, in the from3to3b function. In particular, be aware of invalid characters. (The empty string is not possible since the input is given on the command line.) Also, in order to find the number of significant trits, the leftmost zeros must be discarded.
  • The signature of from3to3b must be char* from3to3b(const char ter[], unsigned int i), where ter is the ternary string and i is the length of ter. The length is computed in main, when checking the input string, and it is useful in from3to3b to know the number of trits, thus how to allocate the output string.
