Summing Integers in a Sequence

Question. The purpose of this exercise is to write an XSLT transform which takes as input a sequence of integers and outputs their sum.

Example. This is an example of input document and this is the corresponding output. You need to download the DTD to run this example.

Hints.
  • The output is text, so you need to set the attribute method="text".
  • You need to declare the type of your parameters, therefore, you need to define the xs (XML Schema) namespace as <xsl:transform ... xmlns:xs="http://www.w3.org/2001/XMLSchema">.
  • In order to select all the children text nodes of an element node num, use num/text().
  • Given a sequence $s, its first item, if any, is $s[1]. If $t is empty, so is $s[1] (it is a sequence as well).
  • In order to cast variable $s to the integer type, use xs:integer($s).
