Converting from 2-complement binary to decimal

Question. The purpose of this exercise is to write a C program which reads on the command-line a string representing a positive integer in 2-complement binary form and outputs its corresponding decimal representation.

Constraints and Hints.
  • The input must be checked in the main function, before the computation takes place, in the from2cTo10 function. In particular, be aware of invalid characters. (The empty string is not possible since the input is given on the command line.)
  • The signature of from2cTo10 must be int from2cTo10(const char bin[]).
  • Reuse the code for converting a number in binary to decimal. What must be changed?

Answer. This is an answer relying on recursive calls.
