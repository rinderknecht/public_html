Converting from balanced ternary to ternary

Question. The purpose of this exercise is to write a C program which reads on the command-line a string representing a positive integer in balanced ternary form and outputs its corresponding balanced ternary representation.

Hints and constraints.
  • The input must be checked in the main function, before the computation takes place, in the from3bto3 function. In particular, be aware of invalid characters and negative numbers. (The empty string is not possible since the input is given on the command line.) Also, in order to find the number of significant trits, the leftmost zeros must be discarded.
  • The signature of from3bto3 must be char* from3bto3(const char ter[], unsigned int i), where ter is the ternary string and i is the length of ter. The length is computed in main, when checking the input string, and it is useful in from3bto3 to know the number of trits, thus how to allocate the output string.
