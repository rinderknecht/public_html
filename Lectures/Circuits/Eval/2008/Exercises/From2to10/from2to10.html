Converting from binary to decimal

Question. The purpose of this exercise is to write a C program which reads on the command-line a string representing an integer in binary form and outputs its corresponding decimal representation.

Constraints.
  • The input must be checked in the main function, before the computation takes place, in the from2to10 function. In particular, be aware of invalid characters. (The empty string is not possible since the input is given on the command line.)
  • The signature of from2to10 must be unsigned int from2to10(const char bin[]).
Answers.
  1. This is a solution using a while loop where the parameter bin is modified directly (since the original value is not needed later).
  2. This is a solution using a for loop. Note how the initialisation part of the loop is empty. If we write instead for (unsigned int dec = 0; *bin ; bin++), it does not work because this way dec is only visible inside the loop and return dec; does not see it after.
  3. This is a solution using a while loop where an integer index idx is used (so bin is unmodified).
  4. This is a solution using recursion (that is, a function calling itself) instead of a syntactic loop. The construct ... ? ... : ...; is a conditional: if the first "..." is true (i.e., is not 0), then the second "..." is executed, otherwise the third "..." is. It is the same as if (...) {...} else {...};.
  5. A variation using a do { ... } while (...) loop.
