Converting from decimal to 2-complement binary

Question. The purpose of this exercise is to write a C program which reads on the command-line a string representing an integer in decimal form and outputs its corresponding 2-complement binary representation.

Hints.
  • The input must be checked in the main function, before the computation takes place, in the from10to2 function. In particular, be aware of invalid characters. (The empty string is not possible since the input is given on the command line.)
  • The signature of from10to2c must be char* from10to2c(int dec).
  • If the number is positive or zero, the problems boils down to the conversion from decimal to binary (make sure to allocate a leftmost zero bit, though). If negative, convert the negated decimal to its binary representation, then consider it as a 2-complement (this is allowed since the number is positive) and negate it, i.e., make the 1-complement and increment it.
