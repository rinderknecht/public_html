Converting from balanced ternary to ternary

Question. The purpose of this exercise is to write a C program which reads on the command-line a number in balanced ternary (with the convention that `I' denoted the trit -1) representation and outputs its corresponding ternary representation.

Constraints.

  • The input must be checked in the main function, before the computation takes place, in the from3bto3 function.
  • In particular, be aware of the nonzero leftmost trit (hence, of the particular case when zero is the input).
  • The signature of from3bto3 must be int* from3bto3(int*).
  • The data structure to encode a number in ternary or balanced ternary is the same: an array of integers, where the first cell (at position 0) holds the number of (following) trits and the remaining cells contain the trits, the most significant being first.
