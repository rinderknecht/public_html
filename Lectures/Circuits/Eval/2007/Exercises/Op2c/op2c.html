Arithmetics on binary 2-complement numbers

Question. The purpose of this exercise is to write a C program which reads on the command-line a binary operation add or mult, followed by two numbers in binary 2-complement notation. The output is the result, in binary 2-complement representation, of the operation on the input numbers.

Constraints.

  • The input must be checked in the main function, before the computation takes place, in the add2c and mult2c functions.
  • The signatures are int* add2c(int*,int*) and in* mult2c(int*,int*).
  • The data structure to encode a number in binary 2-complement is an array of integers, where the first cell (at position 0) holds the number of (following) bits and the remaining cells contain the bits, the leftmost one being first.

Hints.

  • You will need to write first the addition and multiplication on binary numbers.
  • You will need to write the negation of a binary 2-complement number.
  • For debugging purposes, write a function converting from binary 2-complement numbers to decimal numbers (from2cTo10) and a function from binary to decimal (from2To10).
