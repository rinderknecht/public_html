Linear search in arrays

Linear search is a basic technique to search for an item in an array, where the items in the array is compared to the sought item until it is found or all have been examined. The choice of the items for comparison is by increasing indices from the start of the array, or by decreasing indices from the end, hence drawing an imaginary line as in "linear search".

The implementation in Java of this algorithm allows us to review several basic concepts and techniques: empty arrays, index ranges, sequential boolean operators, loops, recursion, sentinels, assertions, loop invariants and termination. This example suggests different test cases: the empty array, the singleton array and the large array. Moreover, the stylistic issues covered are: uniformity of access to array items, the ++ operators, the separation of read and writes (side-effects) in expressions.

For further reading, see Amortized analyses of self-organizing sequential search heuristics (1985) which analyses the costs of two common heuristics whose aim is to improve on linear search.
